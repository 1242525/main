# 🛰️ ARGOS IoT Platform (B2B + OTA)
📌 프로젝트 개요

ARGOS는 기업을 대상으로 IoT 보안 디바이스를 공급하고,
클라우드 기반 OTA(Over-The-Air) 업데이트 및 원격 관리 기능을 제공하는 플랫폼이다.

본 시스템은 단순한 기기 판매를 넘어,
디바이스 운영·보안·업데이트까지 통합 관리하는 B2B IoT 솔루션을 목표로 한다.

🎯 핵심 기능
1. 🏢 B2B 디바이스 공급
기업 고객 대상 IoT 보안 기기 판매
카메라, 센서 등 다양한 디바이스 지원
기업 단위 계정 및 디바이스 그룹 관리
2. 🔄 OTA (Over-The-Air) 업데이트

Over-the-Air Update

원격 펌웨어 업데이트 배포
디바이스 버전 관리
업데이트 스케줄링 및 롤백 지원
대량 디바이스 동시 업데이트

👉 현장 방문 없이 유지보수 가능

3. 📡 디바이스 관리 (Device Management)
디바이스 등록 / 인증
상태 모니터링 (온라인/오프라인)
그룹 기반 관리 (지점, 건물 단위)
로그 및 상태 데이터 수집
